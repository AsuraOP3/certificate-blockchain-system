## 📋 Project Overview
A decentralized certificate issuance and verification system built on Ethereum blockchain. Educational institutions can issue tamper-proof certificates, students can manage them, and employers can verify authenticity instantly without any intermediary.

## ✨ Features
- ✅ **Issue Certificates** - Institutions issue blockchain-secured certificates
- ✅ **Student Dashboard** - Students view all their certificates
- ✅ **Instant Verification** - Anyone can verify without login
- ✅ **QR Code Sharing** - Easy sharing with employers
- ✅ **Blockchain Storage** - Certificate hashes stored on Ethereum
- ✅ **Tamper-Proof** - Any modification breaks the hash

## 🛠️ Technology Stack
- **Frontend**: HTML5, CSS3, JavaScript (No frameworks)
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **Blockchain**: Ethereum (Sepolia Testnet), Solidity
- **Smart Contract**: Custom CertificateContract.sol
- **Development Tools**: Ganache, MetaMask
